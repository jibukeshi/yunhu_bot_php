### 自行部署

请下载机器人主逻辑 `big.php` 和 SDK `sdk.php`，并放到同一目录下。

在 `big.php` 中填入你的 token，并将 `big.php` 的 URL 填入机器人控制台的“配置消息订阅接口”中。

配置完成后，将机器人加入你的群聊，即可开始使用。
